# DevSearch
A platform for developers to connect, share projects, and rate each other's work.
Link:https://devsearch11.herokuapp.com/

## Tech Stack
- Django
- Postgres
- Django REST Framework

## Features
- Share Projects
- Message other developers
- Rate others work
- Search other developers
